⚙️ Configuración Requerida

Este sistema requiere configurar Firebase para almacenar los datos. Siga estos pasos:

  1. Vaya a firebase.google.com y cree un proyecto gratuito
  2. Active "Firestore Database" en modo de prueba
  3. Obtenga la configuración de su proyecto (Config)
  4. Reemplace las credenciales en el código HTML (líneas 17-23)
const firebaseConfig = {
  apiKey: "su-api-key",
  authDomain: "su-proyecto.firebaseapp.com",
  projectId: "su-proyecto",
  ... }
